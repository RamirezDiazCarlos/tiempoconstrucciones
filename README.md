# Tiempo Construcciones

Landing page para Tiempo Construcciones, empresa especializada en construcción con Steel Framing en San Lorenzo, Santa Fe.

## Sobre el Proyecto

Sitio web desarrollado con **Astro 5** que presenta los servicios de construcción en Steel Framing de manera clara y profesional. El diseño minimalista transmite confianza y calidad, características fundamentales en el rubro de la construcción.

## Tecnologías

- **Astro 5** - Framework web moderno y rápido
- **TypeScript** - Tipado estático para mayor confiabilidad
- **Sharp** - Optimización automática de imágenes
- **CSS Custom Properties** - Sistema de diseño consistente

## Características

- ✅ Diseño responsive y mobile-first
- ✅ Optimización automática de imágenes (WebP)
- ✅ Navegación con efecto scroll
- ✅ Secciones desplegables para información detallada
- ✅ Integración con WhatsApp y redes sociales
- ✅ SEO optimizado

## Estructura del Sitio

- **Hero** - Presentación principal con navegación fija
- **Preguntas Frecuentes** - Información clave sobre Steel Framing
- **Servicios** - Detalle de "Llave en Mano" y "Obra Gris"
- **Contacto** - Información de contacto y redes sociales

## Instalación

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview

---

Desarrollado con Astro
```
